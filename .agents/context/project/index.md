# Project Standards & Setup Overview

This directory contains context files for repository standards, infrastructure, and feature tracking.

## Fast Scan Registry

- **`features.md`**: The Logic Ledger. Always consult this file when planning to see what features have already been integrated. Update when features are completed.
- **`conventions.md`**: Naming conventions, Git commit standards, logging rules, and code style. Adhere to these when executing code modifications.
- **`infrastructure.md`**: Workflow setup, CLI commands for startup (`make start`), CI/CD overview, and build deployments.
- **`tech-stack.md`**: Definitions of the tri-node system technologies (Rust/Bevy, Python/SB3, JS/HTML5).
