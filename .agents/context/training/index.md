# Training & AI Overview

This directory contains context files for the Python Macro-Brain, Reinforcement Learning setup, Curriculum, and Bot Behaviors.

## Fast Scan Registry

- **`overview.md`**: RL design principles ("The General"), summary of the 9-stage curriculum, and common curriculum gotchas.
- **`stages.md`**: Deep dive into the mechanics and logic for Stages 0 through 4 (and beyond).
- **`environment.md`**: Detailed breakdown of the Observation Action Space (channels 1-8) and the reward structures (e.g. death penalties, kill rewards, scouting rewards).
- **`bots.md`**: Documentation on hardcoded or scripted bot behavior logic (e.g., TargetSelection, KeepDistance, ObjectiveDefense), primarily how they execute their logic in Python.
